## Install

```bash
pnpm add @starzkg/vuepress-plugin-live2d-widget
```

Or

```bash
yarn add @starzkg/vuepress-plugin-live2d-widget
```

Or

```bash
npm i @starzkg/vuepress-plugin-live2d-widget
```
